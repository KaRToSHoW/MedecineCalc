# Техническое описание

Проект: Медицинский мобильный калькулятор Cockcroft–Gault (Expo + React Native + Firebase).

1) Цель проекта
- Предоставить врачу мобильный инструмент для расчёта клиренса креатинина (Cockcroft–Gault) с историей расчётов, профилем врача и возможностью анализа результатов.

2) Функциональные требования
- Ввод данных пациента: имя, возраст, вес, пол, сывороточный креатинин (с выбором единиц).
- Автоматический расчёт Cockcroft–Gault и сохранение результата в историю (Firestore) при аутентификации врача.
- Профиль врача (имя, клиника, специальность, контакт).
- Экран истории с возможностью фильтрации по пациенту и просмотром детальной записи.
- Дашборд с метриками и трендами.
- Авторизация через Firebase Auth.

3) Нефункциональные требования
- Безопасность: изоляция данных пользователей (правила Firestore). 
- Удобство: мобильная адаптация и быстрый поток ввода данных.
- Надёжность: устойчивость к сетевым ошибкам (корректные сообщения пользователю).

4) Ограничения
- Данные пациентов хранятся только у владельца аккаунта (врач); приложение не реализует сложную систему ролей.

5) Критерии приёмки
- Приложение может рассчитать и сохранить запись в Firestore при авторизации.
- История и дашборд отображают данные, сохранённые в Firestore.

6) Формальное описание пунктов достижения цели
- P1: Возможность создать расчёт и сохранить его — удовлетворяется, если при нажатии «Рассчитать» запис в Firestore создана с полями: `patientName, weightKg, creatinineMgDl, age, female, result, createdAt`.
- P2: Просмотр истории — удовлетворяется, если GET-операция на `users/{uid}/history` возвращает порядок по `createdAt`.
