# Mockups и описание экранов

Ниже — краткие mockup-описания основных экранов. Эти описания можно преобразовать в графические макеты (Figma/Sketch).

1) Calculator (Экран расчёта)
- Поля: `Имя пациента` (опционально), `Вес` (+ выбор kg/lb), `Возраст`, `Креатинин` (+ выбор mg/dL/µmol/L), `Пол`.
- Кнопка `Рассчитать` → вывод результата и автосохранение в историю при авторизации.
- Кнопка/икона для открытия истории пациента.

2) History (История)
- Список карточек: `Имя пациента` (если есть), `Результат`, `Значения и единицы`, `Дата`.
- Поиск/фильтр по имени пациента (опционально).

3) Profile (Профиль врача)
- Поля: `Имя врача`, `Клиника`, `Специальность`, `Контакт`.

4) Dashboard (Дашборд)
- Сегментированный вид: `Обзор`, `Тренды`, `Пациенты`.
- Карточки метрик: всего расчётов, средний GFR, последний результат, маленькие sparklines.

Рекомендации по mockup:
- Размеры: мобильный (iPhone SE / 375pt ширина) и tablet (768pt).
- Цветовая схема: нейтральный фон `#fff`, акцент `#2563eb`, карточки `#f8fafc`.
