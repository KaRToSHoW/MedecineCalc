# Инструкции по включению материалов в отчёт

Файлы, которые следует приложить к отчёту:
- `docs/technical_description.md` — техническое описание и критерии приёмки.
- `docs/architecture.puml` — UML-диаграмма (сгенерируйте картинку через PlantUML и приложите PNG/SVG).
- `docs/mockups.md` — mockup-описания экранов; лучше приложить изображения макетов.
- `docs/data_models.md` — схемы данных и примеры документов Firestore.

Рекомендации:
- Экспортируйте `architecture.puml` в PNG с помощью PlantUML (или онлайн-сервиса) и приложите картинку.
- Для mockups используйте Figma/Sketch/Adobe XD и вложите изображения экранов.
- В отчёте укажите, как запускать проект локально (см. `expo-app/README.md`).
